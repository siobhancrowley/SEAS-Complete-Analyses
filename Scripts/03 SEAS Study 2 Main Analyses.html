To test whether other factors influenced performance on the memory check questions, we created a pair of generalized linear mixed models (GLMMs) to examine the memory check questions for the main trials and those for the comparison trials.

The original model for main trials was rank-deficient; type, siblings, EmQue total, and gender were removed in that order until the model converged–the final model contained trial, interaction (exclusion/inclusion), and age in months. Of these, only age was significant (z = 5.55, p = 2.8^{-8}), indicating that children were more likely to correctly answer the main trial memory check questions as they got older.

The model for the comparison trials also did not converge initially, and siblings, EmQue total, and gender were removed until the model converged–only trial and age were present in this model, and only age was significant (z = 4.1, p = 4.1^{-5}).
